# 甘农卡绑定企业微信付款码操作指引

请按照以下步骤完成绑定。

## 1. 获取付款码链接（企业微信）

1. 打开 **企业微信**。
2. 打开 **一卡通付款码** 页面。
3. 点击右上角 **···**。
4. 选择 **复制链接**。

## 2. 绑定至甘农卡（甘农卡）

![操作步骤03](https://raw.githubusercontent.com/RE-TikaRa/ImgHosting/main/2026年7月5日%20GSAU-Card教程%20(3).jpg)

### 操作步骤 02 ☕️

1. 打开 **甘农卡**。
2. 点击右上角 **+**。
3. 将刚刚复制的链接粘贴到输入框。
4. 点击 **确定** 保存。


---

## 3. 添加桌面应用卡片

1. 返回手机桌面。
2. 根据手机品牌参考下方对应方法。
3. 将甘农卡快捷卡片（小组件）添加到桌面，方便一键打开。

---

## 主流手机添加桌面卡片方法

### 小米 / Redmi

1. 长按桌面空白处。
2. 点击 **添加小部件**（名称可能略有不同）。
3. 搜索 **甘农卡**。
4. 选择对应卡片并添加到桌面。

### OPPO / OnePlus

1. 双指向内捏合桌面（或长按桌面空白处）。
2. 点击 **卡片**。
3. 点击 **全部卡片**。
4. 下滑找到 **插件**。
5. 找到 **甘农卡**，长按拖到桌面。

### vivo / iQOO

1. 双指向内捏合桌面（或长按桌面空白处）。
2. 打开 **原子组件**、**微件** 或 **桌面挂件**。
3. 找到 **甘农卡**。
4. 长按拖到桌面。

---

> [!TIP]
> 不同品牌和系统版本的菜单名称可能略有区别。如果没有找到相关入口，可以在手机 **设置** 中搜索 **桌面小组件**、**服务卡片** 或 **微件** 查看官方说明。
