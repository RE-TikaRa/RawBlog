# 二维数组元素的引用
## 形式
> 数组名[行索引][列索引]

&emsp;&emsp;例如：
```C
 double w[4][2];
// 如下几种写法均正确
//w[0][2]
//w[i][j]
//w[i+k][j+k]

//====错误写法====//
// w[0,2]
// w[i j]
// w(i+k)(j+k)
```
