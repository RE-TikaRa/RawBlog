# 形式

&emsp;&emsp;一般来说，形式类似于：<font color=#20B2AA>表达式1</font><font color=#FF0000>,</font><font color=#20B2AA>表达式2</font><font color=#FF0000>,</font><font color=#20B2AA>表达式n</font>

# 结合性
&emsp;&emsp;从左向右

# 优先级
&emsp;&emsp;<font color=FF0000>所有运算符中，逗号的优先级最低</font>

# 表达式的值
&emsp;&emsp;逗号表达式的值是<font color=FF0000>最后一个表达式的值</font>。

# 考点
1. 逗号运算符的结合性为<font color=FF0000>从左到右</font>。，先计算表达式1，最后计算表达式n，<font color=FF0000>最后一个表达式的值即为该逗号表达式的值</font>。
2. 逗号运算符在所有运算符中，<font color=FF0000>优先级最低</font>。
