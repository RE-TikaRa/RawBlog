# 第二章 前言

咱们先来认识C语言的“ABC”——就像学英语要从26个字母开始一样，编程也要掌握基本要素。

## 从Hello World开始

本章开篇就带大家写最经典的“Hello World”程序。别看它只会打印一句话，这可是每个程序员的成人礼！通过这个小程序，我们会认识C语言的基本框架。

完整示例🌰：
```c
#include <stdio.h>  // 引入输入输出库

int main() {        // 主函数入口
    printf("Hello, World!\n");  // 输出问候语
    return 0;       // 返回成功状态
}
```

## 数据的身份证

接着要认识C语言里的数据类型，就像给不同种类的数据发“身份证”：
- 整数用`int`（比如年龄）
- 小数用`float`/`double`（比如价格）
- 字符用`char`（比如等级）

代码示例🌰：
```c
int age = 20;            // 定义整型变量
float price = 19.99f;     // 定义单精度浮点数
char grade = 'A';         // 定义字符型
```

还会学到怎么给这些数据起名字（变量命名规则），以及如何定义常量——那些程序里不能变的“铁规矩”。

## 运算的秘密

最后会揭秘各种运算符，包括加减乘除这些基础操作，还有自增自减这种C语言特色技巧。重点理解运算优先级，就像知道先穿袜子再穿鞋一样重要！

优先级对比🌰：
```c
int a = 5 + 3 * 2;      // 先乘后加 结果=11
int b = (5 + 3) * 2;    // 加号优先 结果=16
