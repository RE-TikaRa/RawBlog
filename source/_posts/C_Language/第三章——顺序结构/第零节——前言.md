# 第三章 前言

这一章咱们要掌握程序的“基本功”——就像学功夫要先练马步一样重要。

## 程序的三大招式

还记得第一章说的三种基本结构吗？
- **顺序结构**：像排队买奶茶，一个接一个
- **选择结构**：遇到岔路口做选择（后面章节重点）
- **循环结构**：重复执行直到满足条件（后面章节重点）

本章重点修炼第一种招式——顺序结构。虽然它看起来最简单，但却是所有复杂程序的基础。

## 实战训练

我们会学到：
- 怎么让程序输出结果（printf的使用技巧）
- 如何接收用户输入（scanf的正确打开方式）
- 赋值语句的秘密（别搞混了=和==）
- 复合语句的妙用（用{}把多条语句变成一个整体）

代码示例🌰：
```c
#include <stdio.h>

int main() {
    int num1 = 5, num2 = 3;
    int sum = num1 + num2;
    
    printf("第一个数：%d\n", num1);
    printf("第二个数：%d\n", num2);
    printf("两数之和：%d\n", sum);
    
    // 复合语句示例
    {
        int temp = num1;
        num1 = num2;
        num2 = temp;
        printf("交换后：num1=%d, num2=%d\n", num1, num2);
    }
    
    return 0;
}
