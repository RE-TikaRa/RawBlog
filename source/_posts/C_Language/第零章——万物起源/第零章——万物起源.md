# 三、[[安装MSYS2]]
&emsp;&emsp;之前我们安装的VSC只是个编辑软件，将相当于编辑任务清单，具体执行还是需要MSYS来进行配合。
&emsp;&emsp;首先去MSYS2的[官网](https://www.msys2.org/)去下载安装包，网站打开之后点击这个
![image.png](https://s2.loli.net/2025/04/12/AzuaPZJr6WbcnKi.png)
&emsp;&emsp;下载之后的安装包也是一直点下一步即可（如果要修改安装路径，不建议有中文，空格等等），在安装完成之后会打开一个命令行窗口，分别输入以下命令以更新基础数据库。（即一行一行执行）
```MSYS2
pacman -Syu
pacman -Su
pacman -S --needed base-devel mingw-w64-x86_64-toolchain
```
&emsp;&emsp;命令执行期间只要不报错即可，出现[Y/n]的选项都填入Y然后点击回车即可，另说就是，这些命令行无法使用CTRL+C或者+V之类的快捷键，因为在这些系统之中ctrl+c属于停止运行的快捷键，所以一般还是不要用比较好，在界面右键选择copy和paste也可以复制粘贴。
&emsp;&emsp;接下来打开我们刚才的安装路径，比如我是默认安装，就在**C:/msys64**这个路径之下，找到**mingw64**文件夹，然后打开它里面的**bin**文件夹，复制整个文件路径**C:\msys64\mingw64\bin**，在***此电脑图标***上右键，点击***属性***，打开设置之后点击高级系统属性
![image.png](https://s2.loli.net/2025/04/12/LaM5Soh9cOqAdI2.png)
![image.png](https://s2.loli.net/2025/04/12/TniyOB2C3fstHwZ.png)
&emsp;&emsp;在蓝色框里把路径填写进去之后点确定就可以，，然后按住win+r，打开运行窗口，输入CMD，输入如下命令测试安装是否成功。
```CMD
gcc -v
```
&emsp;&emsp;出现如下内容便是安装成功
![image.png](https://s2.loli.net/2025/04/12/VyYk6NSGeL9WtJ1.png)
&emsp;&emsp;至此，我们的基础设置已经全部完成！接下来就是正式学习C语言了。
