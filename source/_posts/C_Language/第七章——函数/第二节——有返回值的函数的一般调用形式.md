# 有返回值函数的调用形式
```C
函数名 （实际参数列表）
```
&emsp;&emsp;    调用的结果是获得一个返回值，该返回值可以参加相应类型的计算。

# 函数调用的过程
1. 首先为被调用函数的所有形参分配内存，再计算实际参数的值，再一一对应地赋给相应的形参（对于无参函数，不做这项工作）。
2. 然后进行函数体，为函数说明部分定义的变量分配存储空间，再依次执行函数体中的可执行语句。
3. 当执行到return语句时，计算返回值，收回本函数中定义的变量所占用的存储空间，对于static类型的变量，其空间不收回），返回主调函数继续执行。

# 函数调用的语法要求
 - 调用函数时，函数名必须与所调用的函数名字完全一致。
 - 实参的个数必须与形参一致。
 - C语言规定，函数必须先定义后调用。
